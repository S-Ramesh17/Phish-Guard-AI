MANDATORY FEATURES:

1. AUTO SCAN
• Scan every webpage automatically on load

2. MANUAL SCAN
• "Scan Now" button in popup
• Re-scan active tab on demand

3. DETECTION LOGIC
• Password input detection
• Non-HTTPS pages
• New domain (WHOIS < 30 days – simulated)
• PhishTank listing (simulated dataset)

4. DETECTION EXPLANATION
• Each scan must store:
  - URL
  - score
  - detected (true/false)
  - list of reasons
  - timestamp

5. REPORT STORAGE
• Store reports in chrome.storage.sync
• Keep last 50 scans

6. POPUP UI
• Status: SAFE / SUSPICIOUS / PHISHING
• List of detection reasons
• Buttons:
  - Scan Now
  - Export Reports (JSON)
  - Open Dashboard
