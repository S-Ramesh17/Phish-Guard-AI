## üöÄ Installation & Usage

### 1. PhishGuard Extension (Chrome/Edge/Brave)
1. Open your browser and go to `chrome://extensions/`
2. Enable **Developer Mode** (top right toggle).
3. Click **Load unpacked**.
4. Select the `extension/` folder from this project.
5. The PhishGuard icon should appear in your toolbar.

**How to use:**
- Visit any website.
- The extension automatically scans the page.
- Click the icon to see the Security Score and Status.
- Click "Scan Now" to force a re-scan.
- Click "Export Reports" to download the `.json` log file.

### 2. PhishGuard Hub (Website)
1. This project is hosted on Replit.
2. The website serves the `website/` directory content.
3. Open the Dashboard.
4. Click **Import Extension JSON** and upload the file you exported from the extension.
5. View detailed analytics and export to PDF.
